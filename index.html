About Apatar

Apatar is a data integration tool created in Java that is open-sourced, and is currently capable of importing data from multiple database systems.

Apatar uses a visual mapping tool to make connections to database sources, transforming the data, and to set up a unified schema for providing a single view of the data integrated.

Data Sources Apatar can integrate

Apatar is capable of integrating data from various database systems, such as :

  • Oracle
  • MS SQL
  • MySQL
  • Sybase
  • DB2
  • MS Access
  • PostgreSQL
  • XML
  • MS Excel
  • ... any JDBC data sources and more
